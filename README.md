# 입고 예약 보드 v0.20

## 반영 내용
- 상단 로고 다시 적용
- 로고를 제목과 같은 라인에 더 크게 표시
- 브랜드 선택 필터 유지: 오늘/내일 라인 오른쪽 `브랜드 전체`
- 고객사/발주번호 검색 유지
- 시트 M열 층 기준 유지
- 전체/완료/미입고/승인대기 필터 유지
- 컨테이너 문구만 제거: `컨테이너 20FT` → `20FT`
