# 입고 예약 보드 - 현장용

QUERY 구글시트를 바로 읽는 현장용 보기 페이지입니다.

## 이번 버전 변경점
- 브랜드별 구분 → 층별 구분(M열 층 기준)
- 컬럼 순서 변경: 시간 / 발주번호 / 톤수 / 작업 / 메모 / 상태 / 고객사
- 시트의 `업체명`은 화면에서 `고객사`로 표시
- Shadcn 스타일을 참고한 카드/버튼/배지 디자인으로 정리
- 모바일 화면에서 컬럼이 깨지지 않도록 카드형 행 레이아웃 적용

## 기능
- 오늘 / 내일 탭
- 전체 / 완료 / 미입고 / 승인대기 현황
- 층별 카드
- 완료 9 / 12 표시
- 취소 숨김
- 완료 / 승인대기 상태 표시
- 검색(고객사/발주번호)
- 작은 새로고침
- 메모 칸 항상 표시
