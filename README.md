# 입고 예약 보드 - 현장용 v0.13

## 변경 내용
- 상단 현황 카드 클릭 필터 추가
  - 전체: 전체 입고
  - 완료: 완료 건만
  - 미입고: 완료/승인대기가 아닌 건
  - 승인대기: 승인대기 건만
- 행 컬럼에 `층` 다시 추가
- 완료 행은 회색 음영 처리 유지 및 강화
- 모바일 한 줄 표 + 가로 스크롤 유지

## v0.12 변경
- 모바일에서 옆으로 밀지 않고 2줄 레이아웃으로 표시
- 1줄: 시간 / 발주번호 / 고객사 / 상태
- 2줄: 톤수 / 작업 / 메모
- 층은 카드 상단 제목으로 유지하고, 모바일 행 안에서는 숨김
- 발주번호와 톤수 영역 간격 조정
